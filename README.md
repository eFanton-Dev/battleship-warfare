# battleship-warfare
This project is an offline battleship game that can be played in 2 people. The gameplay is divided in 2 pahses: positioning and attacking.
- Positioning: each player has to select the ships on the center and place them in the game area (if the square become gray then the ship of that kind are finished)
- Attacking: each player can try to hit the center of each ship, wheb that happen the ship will sink and disappear from the game area

One of the player win when they manage to destroy all the enemy ships.

This game created for the Daydreams event in Padua. Such event consisted in a 8 hour long competition during which a group of 4 had to create a game.
